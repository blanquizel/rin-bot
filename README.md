# rin-bot
出于个人兴趣编写的bot。

+ 基于Koishi和go-cqhttp库
+ 仅接入onebot频道
+ 参考[2bot](https://github.com/idlist/2bot-v4)


### 当前支持指令
+ 抽卡模拟
    ```
    mrfz-1 // 明日方舟卡池单抽
    mrfz-10 // 明日方舟卡池十连
    ```

+ 摸鱼皇帝
    ```
    今日摸鱼榜
    昨日摸鱼榜
    ```

+ 法定假期查询
    ```
    剩余假期
    ```
